My Restaurant Project CRUD API

This API allows you to manage restaurants in the system.

Navigate to the following /api-docs/#/ extention to get acess to swagger UI.

Routes

  • GET /restaurants

    Get all restaurants in the system.

  • GET /restaurants/:id

    Get a restaurant by ID.

  • POST /restaurant

    Create a new restaurant in the system.

    Parameters:

    • name: Name of the restaurant (string, required)

  • PUT /restaurant/:id

    Update an existing restaurant.

    Parameters:

    • name: Name of the restaurant (string, required)

  • DELETE /restaurant/:id

    Delete a restaurant by ID.
